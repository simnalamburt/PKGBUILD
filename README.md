PKGBUILD
========
Homemade Arch Linux packages for personal needs.

- [libsndfile](libsndfile)

If you don't want to install pre-built packages, you can use [`makepkg`] to
build and install these packages manually.

```bash
cd <package>
makepkg -sri
```
